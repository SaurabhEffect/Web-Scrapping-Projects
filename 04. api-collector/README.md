# REST Countries API Data Collector

[![Python](https://img.shields.io/badge/Python-3.x-blue.svg)](https://python.org)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Code style: black](https://img.shields.io/badge/code%20style-black-000000.svg)](https://github.com/psf/black)

A Python-based data collector that fetches, flattens, and cleans extensive country data from the free [REST Countries API](https://restcountries.com). 

This tool efficiently produces a comprehensive, tidy dataset featuring 250 countries with over 15 distinct data points per country — such as population, area, currencies, languages, and timezones — which is perfectly structured for immediate use in machine learning workflows, data analysis, or BI dashboards.

---

## Features

- **Automated Flattening**: The REST Countries API originally returns deeply nested JSON structures. This application safely extracts and flattens essential fields to yield flat rows without data loss.
- **Data Engineering**: Implements rigorous type-casting of numeric columns, handles nulls or completely missing keys gracefully, and calculates derived metrics (such as population density per km²).
- **Multiple Exports**: Effortlessly generates clean exports in both `CSV` and `JSON` formats for ultimate versatility (e.g., pandas, SQLite, PostgreSQL, Excel).
- **Built-in Resilience**: Includes full retry logic with backoff for connection timeouts or HTTP errors to ensure reliability even over unstable internet connections.
- **Detailed Logging**: Logs events to both the console and a local timestamped log file (`api_collector.log`) to keep track of any failures or anomalies.

---

## Tech Stack

- **[requests](https://pypi.org/project/requests/)**: Simplifies RESTful API calls and HTTP requests handling.
- **[pandas](https://pandas.pydata.org/)**: Powers dataset manipulation, deduplication, filling missing data, sorting, and seamless multi-format exports.
- **[json](https://docs.python.org/3/library/json.html)**: Handles raw nested JSON parsing implicitly.
- **[logging](https://docs.python.org/3/library/logging.html)**: Provides a structured logging approach out-of-the-box.

## Setup & Execution

### Prerequisites

Ensure you have Python 3.8+ installed. You can download it from [python.org](https://www.python.org/).

### Installation Steps

1. **Clone the repository and navigate into the project directory:**
   ```bash
   git clone <repository-url>
   cd api-collector
   ```

2. **Set up a virtual environment (recommended):**
   - **Windows:**
     ```bash
     python -m venv venv
     .\venv\Scripts\activate
     ```
   - **Linux / macOS:**
     ```bash
     python3 -m venv venv
     source venv/bin/activate
     ```

3. **Install the required packages:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Run the collector:**
   ```bash
   python collector/api_collector.py
   ```

> **Note**: This script uses the public REST Countries API which does not require an API key and is completely free.
